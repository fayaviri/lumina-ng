# Responsive Guidelines

Lumina NG must work well on desktop, tablet and mobile.

## Rules

- Tables should use PrimeNG responsive behavior or provide a mobile-friendly alternative.
- Dialogs should use widths such as `min(92vw, 30rem)` or breakpoints.
- Form grids should collapse to one column on small screens.
- Sidebar behavior must be tested on mobile.
- Avoid fixed widths that overflow small screens.
