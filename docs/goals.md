# Lumina NG Goals

Lumina NG is a free Angular + PrimeNG Aura admin template created to be reusable, modern, responsive and easy to modify with AI assistance.

## Main Priorities

1. Balanced input and text sizes.
2. Simple structure for AI-assisted editing.
3. Responsive behavior for desktop and mobile.
4. Angular 21 modern patterns.
5. Generic template examples, not project-specific business logic.

## What Lumina NG Is

A base template for administrative web applications.

## What Lumina NG Is Not

A finished system for a specific business domain.
